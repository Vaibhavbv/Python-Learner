## ✨ Features
* **60 Interactive Topics:** 10 fully loaded deep-dives and 50 structured roadmap skeletons.
* **Immersive 3D Hero Section:** Powered by `Three.js` and `@react-three/fiber` for a premium look.
* **Fluid Animations:** Butter-smooth transitions handled by `Framer Motion`.
* **Dark & Light Modes:** Because every developer needs options. (Theme persists across sessions!)
* **Deep Search & Filters:** Instantly find what you need with global search (`/` hotkey to focus) and importance filters (Must Know, High, Medium, Low).
* **Local Progress Tracking:** Automatically saves your completed topics right into your browser (`localStorage`).
* **Built Different Aesthetics:** Custom syntax highlighting, terminal-meets-editorial UI, and beautifully animated progress bars.

This project was entirely rebuilt into a modern SPA architectured for speed and visual excellence:
- **Framework:** React 18 + Vite 5
- **Styling:** Tailwind CSS v3 + Custom CSS Variables
- **Animations:** Framer Motion v11
- **3D Graphics:** Three.js r165 + `@react-three` ecosystem
- **Syntax Highlighting:** highlight.js

## 🚀 Getting Started

Want to run the roadmap locally? It's as simple as:

### 1. Clone the repository
```bash
git clone https://github.com/Vaibhavbv/Python-Learner.git
cd Python-Learner
```

### 2. Install dependencies
```bash
npm install
```

### 3. Spin up the dev server
```bash
npm run dev
```

The app will be running at `http://localhost:5173`. 

---

## 📦 Deployment (Vercel)
This project is pre-configured and 100% production-ready for **Vercel**.
1. Import the repository into your Vercel dashboard.
2. Vercel automatically detects the Vite framework.
3. Keep default build settings (`npm run build` and `dist` output directory).
4. Click **Deploy**! 🚀
